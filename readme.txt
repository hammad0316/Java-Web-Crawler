To get the program to run:
1) Make sure the userDB.txt and itemDB.txt are still in the same folder the files are in
2) Open cmd prompt and get into the directory that contains the files
3) Type 'javac Phase3.java'
4) Then to run type 'java Phase3'
	a. If you'd like to run with a file, type '-i FILENAME' in which FILENAME is the name of the file you want to input the data from 
	b. If you'd like a specific output file, type '-o OUTFILE' in which OUTFILE is the name of the file you want the data to be outputted to
5) Admin login username is 'Admin' and password is 'password'
6) Any account you make is going to be the one right under admin and won't be able to see everyone else's searches
